# Welcome — I'm Devin Kane

I'm a Computer Science student at the University of Illinois Chicago focused on building practical, full-stack software with clean systems design, reliable tooling, and usable interfaces.

My work is driven by the same mindset that first pulled me toward building: turn ideas into something useful, tangible, and polished. I enjoy projects that connect lower-level systems, developer tooling, and user-facing applications — from Rust-based desktop utilities to Go/React web apps and C command-line tools.

Currently, I work as a Computer Specialist at UIC, supporting campus users across software, hardware, networking, and OS-related issues. I also previously contributed to OS and systems research involving secure I/O and kernel-level work.
